# Контроль потребления концентратов

Протокол: Теленок с 0-2 месяцев
День от начала: 30

Коротко:
Проверить потребление комбикорма. Снимать с молока можно только при потреблении не менее 500г концентратов в сутки, лучше при 1кг

Препарат: Весы
Дозировка: Контроль минимум 500г/сутки
Способ применения: Манипуляция

> Примечание: данные из вашей базы Notion. Сверять с инструкцией и ветврачом.
