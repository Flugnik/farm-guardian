# Начать подкормку комбикормом

Протокол: Теленок с 0-2 месяцев
День от начала: 4

Коротко:
Начать подкормку комбикормом по специальному рецепту. Комбикорм должен быть постоянно в доступе. Цель: привес 750-800 г/сутки!

Препарат: Комбикорм по рецепту
Дозировка: Кукуруза 50% + Овёс 27,9% + БВМД 20%  или соя экструдированная + Патока 2% + Премикс 0,1%
Способ применения: Перорально

> Примечание: данные из вашей базы Notion. Сверять с инструкцией и ветврачом.
